\section{ Java }
Java es un lenguaje de propósito general desarrollado por Sun Microsystems (después adquirida por Oracle) que reúne las siguientes características:
\begin{itemize}
 \item Compilado
 \item Orientado a objetos
 \item Basado en clases
 \item Concurrente 
\end{itemize}

\section{ Java EE }
